Value Mappings são um De x Para de valores que se pode invocar ao longo de uma integração.

Criando um Value Mapping

- Acesse seu Integration Package
- Clique em 'Add'
- Value Mapping

Na parte de cima cada Value Mapping é representado por uma linha em que:
- 'Agency' é o nome do sistema, dando no "De" quando no "Para"
- 'Identifier' é o nome do campo que queremos mapear

Na parte de baixo temos o "De x Para" e devemos criar os valores que iremos receber e para qual valor queremos transformá-lo
